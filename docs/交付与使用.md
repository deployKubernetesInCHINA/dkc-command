交付使用说明
--

1. 此项目发布1个文件, koictl.zip
   解压之后,项目结构包含以下内容. koictl-xxxx是各平台的可执行文件,inventory是拓扑文件目录
   ```
   $ unzip koictl.zip -d koictl
   $ tree -L 1
    .
    ├── README.md
    ├── docs
    ├── images
    ├── inventory
    ├── koictl-darwin
    ├── koictl-linux-amd64
    ├── koictl.exe
    ├── static
    └── views
    
    5 directories, 4 files
   ```
2. 在内网, 运行可执行文件的download命令,下载离线文件之后,
   当前文件结构包含了全部的离线安装文件. 可以将此项目任意移动到需要部署k8s集群的服务器上进行安装操作
   ```
   $ ./koictl-linux-amd64 download
   $ tree -L 1
    .
    ├── ansible
    ├── centos7_9_2009
    ├── centos8_3_2011
    ├── docs
    ├── images
    ├── inventory
    ├── k8s-offline-install
    ├── koictl-darwin
    ├── koictl.exe
    ├── koictl-linux-amd64
    ├── koictl.zip
    ├── kubespray_cache
    ├── README.md
    ├── static
    └── views
    
    10 directories, 5 files
   ```

3. 编辑拓扑文件inventory/hosts.yaml,查看拓扑结构
   ```
   $ ./koictl-linux-amd64 prepare -s
        HOSTNAME       MASTER   NODE     MONGO     OM    SG         IP                       Release
    ------------------ -------- ------ ----------- ----- ----- --------------- --------------------------------------
     surpass-master-1   yes                                     192.168.57.36   CentOS Linux release 7.9.2009 (Core)
     surpass-master-2   yes                                     192.168.57.37   CentOS Linux release 7.9.2009 (Core)
     surpass-master-3   yes                                     192.168.57.38   CentOS Linux release 7.9.2009 (Core)
     surpass-worker-1            yes    arbiter           yes   192.168.57.39   CentOS Linux release 7.9.2009 (Core)
     surpass-worker-2            yes                      yes   192.168.57.40   CentOS Linux release 7.9.2009 (Core)
     surpass-worker-3            yes                yes   yes   192.168.57.41   CentOS Linux release 7.9.2009 (Core)
     surpass-worker-4            yes    primary                 192.168.57.42   connect err.
     surpass-worker-5            yes    secondary               192.168.57.43   connect err.
   ```
4. 安装k8s/mongo
   ```
   $ ./koictl-linux-amd64 install -k
   $ ./koictl-linux-amd64 install -m
   ```
